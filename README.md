## Usage

Run the main script with a topic of your choice.

**Example:**
```bash
python main.py "US vs Venezuela Tension"
```

If you run it without arguments, it will ask you to enter the topic interactively:
```bash
python main.py
```

## Architecture

The system follows this workflow:
1. **Human Feedback**: Asks for clarification on the topic.
2. **Researcher Agent**: Iteratively searches the web (using DuckDuckGo) to gather information, up to 5 iterations.
3. **Outline Generation**: Creates a blog post outline and list of sections based on research.
4. **Section Writers**: Generates content for each section in parallel.
5. **Compilation**: Compiles all sections into a final markdown report.

## Files

- `main.py`: Entry point.
- `graph.py`: Graph definition using LangGraph.
- `nodes.py`: Implementation of graph nodes (agents).
- `state.py`: State definitions.
- `tools.py`: Search tool wrapper.
